Configurações

Descrição

Na tela de administração, você poderá atualizar as informações diárias do cardápio escolar, como a merenda, o almoço e as informações nutricionais. Para modificar essas informações, é necessário ter permissão de administrador. Para autorizar o acesso, basta clicar no botão "Autorizar". Apenas usuários com privilégios de administrador terão permissão para editar os dados do cardápio, garantindo que o cardápio Onrange esteja sempre atualizado com as refeições do dia para os alunos.
